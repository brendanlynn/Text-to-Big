# Text-to-Big

A console application that converts small (normal-sized) text such as

```
HELLO
```

to large (ASKII-art) text such as

```
#   # ##### #     #      ###
#   # #     #     #     #   #
##### ##### #     #     #   #
#   # #     #     #     #   #
#   # ##### ##### #####  ###
```

. It's that simple.
